| tab shell -> product expectations | Early shell choices can lock in misleading or unstable navigation if implemented loosely |

| T-01-07 | D | launch routing | mitigate | Fail safely to the privacy screen when acknowledgement state cannot be read; keep all gate dependencies local-only |
