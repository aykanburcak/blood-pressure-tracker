## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| app bootstrap -> local runtime config | Untrusted config mistakes or third-party SDKs can turn launch into a network-dependent or privacy-breaking path |
| privacy gate state -> local persistence | A bad flag contract can let the app skip required first-run disclosure or strand the app on launch |
| test harness -> execution feedback | Missing or broken automation would hide regressions in a trust-critical shell |
